AdSense Page Level Ads for WordPress done right.

== Description ==
Easily integrate AdSense Page Level Ads into your WordPress Site. Just enter your Publisher ID and choose where to display Page Level Ads!

== Installation ==
1. Upload the `adsense_page_level_ads` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. Configure the plugin through the `Settings` => `Page Level Ads` menu in the WordPress
